# FocusTimer — Coding Interview Project (iOS / Swift)

A Swift Package for the coding interview session.

## Quick start

```
swift build
swift test
```

## Project layout

```
Sources/FocusTimerAnalytics/
├── Model/
│   └── FocusSession.swift        (data model)
├── Repository/
│   └── FocusRepository.swift     (data source)
└── Analytics/                    (empty — for your work)

Tests/FocusTimerAnalyticsTests/
└── (tests live here)
```

Package manifest: `Package.swift` (swift-tools-version 5.9).

## AI tools

Connect your preferred tools to the interview pod:

- **VS Code / Cursor**: Open locally → Remote Explorer → connect via SSH
- **Claude Code**: SSH into the pod from your terminal

## Task

Your interviewer will brief you on the task at the start of the session. No task file is included in this repo.
